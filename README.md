Hospital Management System
This project presents a Hospital Management System designed to streamline hospital operations and improve healthcare services.
It manages patients, doctors, nurses, departments, rooms, medical records, emergencies, ambulances, and staff data efficiently.

Key Features:

Patient registration and medical record management.

Staff, doctor, and nurse information management with multi-phone support.

Department and room organization with availability tracking.

Emergency case handling and ambulance management.

Drug prescriptions linked to patient records.

Data integrity maintained through a relational database with foreign keys and cascading rules.

Sample data insertion and several SQL queries for reporting and analysis.

The project includes:

ERD (Entity Relationship Diagram).

Database schema design with normalization.

SQL scripts for table creation, data insertion, and advanced queries.
